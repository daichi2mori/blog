### ログイン時自動起動

```bash
dropbox autostart [y/n]
```

- y: ログイン時に自動的に開始します
- n: ログイン時に自動的に開始しません


### 同期設定

現在同期の対象にしていないディレクトリのリストを表示します

```bash
dropbox exclude
```

除外リストにディレクトリを追加しDropboxを再び同期します

```bash
dropbox exclude add [DIRECTORY] [DIRECTORY] …
```

除外リストからディレクトリを削除し、Dropboxを再び同期します

```bash
dropbox exclude remove [DIRECTORY] [DIRECTORY] …
```


### ファイルステータス

```bash
dropbox file status [-l] [-a] [FILE]…
```

ファイルの現在のステータスを表示します

- -l or --list: Linuxコマンドlsに似た形式で情報を表示します
- -a or --all:  「.」で始まるエントリを無視しません


### ヘルプ

```bash
dropbox help [command]
```

引数がない場合、コマンドのリストと各コマンドの簡単な概要を表示します。
コマンドがある場合はコマンドの使用方法を表示します。


### LAN同期

```bash
dropbox lansync [y/n]
```

- y: LAN同期を使用します（デフォルト設定）
- n: LAN 同期を使用しません


### プロキシ

```bash
dropbox proxy MODE [TYPE] [HOST] [PORT] [USERNAME] [PASSWORD]
```

- TYPE: https、socks4、socks5のうちのどれか1つ
- HOST: プロキシのホスト名
- PORT: プロキシのポート
- USERNAME: プロキシのユーザー名（オプション）
- PASSWORD: プロキシのパスワード（オプション）


### デーモン

デーモンを実行中は 1、実行していない場合は 0 を返します

```bash
dropbox running
```

Dropbox デーモンを開始します。デーモンをすでに実行している場合は現状が維持されます。システムで利用不可の場合、Dropbox デーモンを自動インストールするため、オプションに [-i] または [--install] が含まれます。

```bash
dropbox start [-i]
```

デーモンを停止します

```bash
dropbox stop
```


### ステータス

デーモンの現在のステータスを表示します

```bash
dropbox status
```


### バンド幅

```bash
dropbox throttle DOWNLOAD UPLOAD
```

- DOWNLOAD: 無制限または手動モードの上限を KB/秒で設定
- UPLOAD:   無制限、自動または手動で KB/s を設定
