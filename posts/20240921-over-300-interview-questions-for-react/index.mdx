### React とは？

React はオープンソースのフロントエンド JavaScript ライブラリで、特にシングルページアプリケーションのユーザーインターフェイスを構築するために使用される。ウェブアプリやモバイルアプリのビューレイヤーを処理するために使用される。React は、Facebook で働くソフトウェアエンジニア、ジョーダン・ウォルケによって作られた。React は 2011 年に Facebook のニュースフィードに、2012 年には Instagram に初めて導入された。

### React の主な特徴は？

・直接リアル DOM を操作するのはコストが高いため、代わりに仮想 DOM を使用している。

・サーバーサイド・レンダリングをサポートしている。

・一方的なデータフローまたはデータバインディングに従う。

・ビューを開発するために、再利用/再構成可能な UI コンポーネントを使用する。

### JSX とは？

JSX（JavaScript XML）は、ECMAScript の XML ライクな構文拡張です。基本的には、React.createElement()関数のためのシンタックスシュガーを提供するだけで、HTML のようなテンプレート構文とともに JavaScript の表現力を与えてくれます。
